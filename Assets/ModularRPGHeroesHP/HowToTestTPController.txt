Before you test TPController, you have to these things.

1. Go to Edit / Project Settings / Player /Active Input Handling. Make sure it is set as "Both" or "Input Manager(Old)".

2. Go to Edit / Project Settings / Input.  Go to Upper Right corner of window, and click the button in the middle for "Select Preset"

3. Select "Input Manager" in the list.

4. Go to Physics. do the same thing for "Physics Manager".

You are good to test!
